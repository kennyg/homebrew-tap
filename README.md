# Homebrew Tap

Custom Homebrew tap for tools by [@kennyg](https://github.com/kennyg).

## Installation

```bash
brew tap kennyg/tap
```

## Available Formulae

### tome

AI Agent Skill Manager - Your spellbook for AI agent capabilities.

```bash
brew install kennyg/tap/tome
```

Or directly:

```bash
brew install kennyg/tap/tome
```

## Updating

```bash
brew update
brew upgrade tome
```
